# Mate — 컴포넌트 라이브러리

> Mate 프로젝트 전체 페이지에서 공통으로 사용되는 UI 컴포넌트 정의서입니다.
메인 페이지 HTML + 전체 와이어프레임 기반으로 작성되었습니다.
> 

---

## 1. 디자인 토큰

### 색상

```
Primary(보라)         #6C63FF   → 버튼, 로고, 강조, 태그
Primary-soft(연보라)  #EDE9FF   → 태그 배경, 뱃지 배경
Accent(핑크)          #FF6B9D   → 포인트 색상
Background            #EEF2F8   → 페이지 배경
Surface(흰색)         #FFFFFF   → 카드, 입력창 배경
Text-primary          #1A1A2E   → 제목, 본문
Text-secondary        #6B7280   → 부제목, 설명
Text-muted            #9CA3AF   → 날짜, 힌트
Border                #E5E7EB   → 구분선, 테두리
```

### 상태 색상

```
모집중(초록)    #DCFCE7 배경 / #166534 텍스트
마감임박(노랑)  #FEF3C7 배경 / #92400E 텍스트
마감됨(회색)    #F3F4F6 배경 / #6B7280 텍스트
```

### Border-radius

```
카드           16px
버튼/입력창    10px
뱃지/태그/pill 99px
```

## 2. 공통 컴포넌트

### 2.1 Header

모든 페이지 최상단 고정 (`position: fixed`)

```
┌──────────────────────────────────────────────────────────────┐
│ [mate]  [Explore] [Post] [Community]      [로그인] [Get Started] │
└──────────────────────────────────────────────────────────────┘
```

| 요소 | 설명 | 비고 |
| --- | --- | --- |
| `nav-logo` | "mate" 텍스트 로고 | 보라색, 굵게 |
| `nav-links` | Explore / Post / Community | 페이지 이동 |
| `btn-ghost` | 로그인 버튼 | 테두리 없음 |
| `btn-primary` | Get Started 버튼 | 보라 배경 |

**로그인 상태별 변화:**

```
비로그인:  [로그인]  [Get Started]
로그인 후: [👤 Jina_P ∨]           ← 닉네임 + 드롭다운
```

---

### 2.2 Footer

모든 페이지 최하단

```
┌──────────────────────────────────────────────────────────────┐
│  ⓒ 2026 Mate. All rights reserved.                          │
└──────────────────────────────────────────────────────────────┘
```

---

### 2.3 Breadcrumb (경로 표시)

페이지 상단에 현재 위치 표시

```
🏠 > Explore > React 심화 스터디 모집합니다
🏠 > Explore > 모집글 수정
🏠 > My Projects > AI 주식 추천 스터디 > 팀 게시판
```

**사용 위치:** 상세 페이지, 모집글 수정, 팀 게시판

---

### 2.4 Button (버튼)

```
Primary (기본)
┌──────────────┐
│   지원하기    │  → 보라 배경 #6C63FF, 흰색 텍스트
└──────────────┘

Secondary (보조)
┌──────────────┐
│    북마크     │  → 흰색 배경, 회색 테두리
└──────────────┘

Ghost (텍스트형)
   로그인         → 배경 없음, 테두리 없음, 회색 텍스트

Danger (위험)
┌──────────────┐
│  게시글 삭제  │  → 빨간 배경, 빨간 테두리, 흰색 텍스트
└──────────────┘

Disabled (비활성)
┌──────────────┐
│  마감됨(비활성)│  → 회색 배경, 회색 테두리, 클릭 불가
└──────────────┘
```

**사용 위치별 정리:**

| 버튼명 | 타입 | 사용 페이지 |
| --- | --- | --- |
| Get Started | Primary | 헤더 (전체) |
| 지원하기 | Primary | 상세 페이지 |
| 검색 | Primary | 메인 페이지 |
| 댓글 등록 | Primary | 상세 페이지 |
| 글쓰기 | Primary | 팀 게시판 |
| 작성완료 | Primary | 모집글 작성 |
| 수정완료 | Primary | 모집글 수정 |
| 모집 조기마감 | Secondary | 모집글 수정 |
| 취소 | Ghost | 모집글 수정 |
| 관심 저장 / 북마크 | Secondary | 상세 페이지 |
| 게시글 삭제 | Danger | 모집글 수정 |

### 2.5 Badge / 상태 뱃지

```
[모집중]    → #DCFCE7 배경 / #166534 텍스트
[마감임박]  → #FEF3C7 배경 / #92400E 텍스트
[마감됨]    → #F3F4F6 배경 / #6B7280 텍스트
[방장]      → #EDE9FF 배경 / #6C63FF 텍스트
[멤버]      → #F3F4F6 배경 / #6B7280 텍스트
[공지]      → #EDE9FF 배경 / #6C63FF 텍스트  ← 팀 게시판 공지글
```

**사용 위치:** 모집글 카드, 상세 페이지, 멤버 목록, 팀 게시판

---

### 2.6 Tag / 기술스택 태그

```
[React] [Spring] [Python] [Figma] [TypeScript] ...
→ 모든 기술스택 태그 보라색 하나로 통일
→ 배경: #EDE9FF / 텍스트: #6C63FF
→ 사용자가 모집글 작성 시 자유롭게 추가 가능 (최대 10개, 20자 이내)
```

**사용 위치:** 모집글 카드, 상세 페이지, 모집글 작성/수정 페이지

---

### 2.7 Avatar (아바타)

프로필 이미지가 없을 때 닉네임 첫 글자를 동그라미로 표시

| 크기 | 지름 | 사용 위치 |
| --- | --- | --- |
| 크게 | 48px | 마이페이지 프로필 |
| 중간 | 32px | 댓글 목록, 팀 멤버 목록 |
| 작게 | 24px | 모집글 카드 멤버 스택 |

**색상:**

```
av1 → #6C63FF (보라)    av2 → #FF6B9D (핑크)
av3 → #34D399 (초록)    av4 → #F59E0B (노랑)
```

**동작 방식:**

- 프로필 이미지 있으면 → 이미지 표시
- 프로필 이미지 없으면 → 닉네임 첫 글자 + 색상 동그라미 표시

---

### 2.8 FormInput (입력창)

로그인, 회원가입, 모집글 작성/수정 등 입력이 필요한 모든 페이지 공통

```
기본   ┌──────────────────────────┐
       │  placeholder...          │  → 회색 테두리
       └──────────────────────────┘

포커스 ┌──────────────────────────┐
       │  입력중...               │  → 보라 테두리
       └──────────────────────────┘

성공   ┌──────────────────────────┐
       │  입력값                  │  → 초록 테두리
       └──────────────────────────┘
       ✓ 사용 가능합니다

에러   ┌──────────────────────────┐
       │  입력값                  │  → 빨강 테두리
       └──────────────────────────┘
       ✗ 에러 메시지
```

**사용 위치:** 로그인, 회원가입, 모집글 작성/수정, 댓글 입력

---

### 2.9 Pagination (페이지네이션)

```
[◀]  [1]  [2]  [3]  ...  [▶]
      ↑ 현재 페이지 → 보라색 강조
```

**사용 위치:** 메인 페이지 목록, 팀 게시판 목록

---

### 2.10 CommentItem (댓글 아이템)

```
(김) 김영희  1시간 전
     안녕하세요! 참여하고 싶습니다.
     수정  삭제                    ← 본인 댓글에만 표시
```

**사용 위치:** 모집글 상세 페이지, 팀 게시판 (추후 구현)

---

### 2.11 Toast Message (토스트 알림)

화면 우측 하단에 잠깐 표시

```
✓  지원이 완료되었습니다     → 초록
✗  오류가 발생했습니다       → 빨강
!  로그인이 필요합니다       → 노랑
✓  수정이 완료되었습니다     → 초록
✓  게시글이 등록되었습니다   → 초록
```

**사용 위치:** 모든 페이지 공통

---

### 2.12 Skeleton / Empty State (로딩 / 빈 상태)

**Skeleton (로딩 중):**

```
┌─────────────────────────────────┐
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  │
│ ░░░░░░░░░░                      │
│ ░░░░░  ░░░░░  ░░░░░            │
└─────────────────────────────────┘
```

API 호출 중일 때 실제 화면 모양과 비슷하게 회색 박스로 미리 채워서 표시

**Empty State (데이터 없음):**

```
       (•‿•)
  검색 결과가 없습니다
  다른 키워드로 검색해보세요

       (•‿•)
  아직 게시글이 없습니다
  첫 번째 글을 작성해보세요       ← 팀 게시판용
```

**사용 위치:** 모든 페이지 공통
